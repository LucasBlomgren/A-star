# A-star

----Hotkeys----

[Left-click] = Draw obstacle

[Right-click] = Delete obstacle

[S] = Assign start-location

[E] = Assign end-location

[Q] = Drawing after search / while search is ongoing

[A] = Manhattan / Euclidean distance

[SPACE] = Restart algorithm
